# Slides Assets (4311102) — Codex

This folder contains unit‑wise SVG diagrams for Fundamentals of Electronics (4311102).

Usage in Markdown/Slidev:

- Markdown: `![](./unit-2/bridge-rectifier.svg)`
- Slidev (Markdown): `![](/content/resources/study-materials/11-ec/sem-1/4311102-funda-electronics/codex/unit-2/bridge-rectifier.svg)`

Tips

- Prefer relative paths when slides live near this repo, or absolute repo paths when building from elsewhere.
- Gujarati variants use the `.gu.svg` suffix, e.g., `bridge-rectifier.gu.svg`.
- Index launchers: open `unit-map.svg` or a unit’s `index.svg` to browse.
